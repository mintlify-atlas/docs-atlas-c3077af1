`RadarInitializeOptions` provides configuration options for initializing the Radar SDK, including notification handling and push notification settings.

## Properties

<ResponseField name="autoLogNotificationConversions" type="boolean" required>
  A boolean indicating whether to automatically log notification conversions. When enabled, the SDK will track when users interact with notifications.
</ResponseField>

<ResponseField name="autoHandleNotificationDeepLinks" type="boolean" required>
  A boolean indicating whether to automatically handle deep links from notifications. When enabled, the SDK will process and navigate to deep links contained in notifications.
</ResponseField>

<ResponseField name="silentPush" type="boolean" required>
  A boolean indicating whether to use silent push notifications. When enabled, notifications are delivered silently without alerting the user.
</ResponseField>

## Example

```swift
let options = RadarInitializeOptions(dict: [
    "autoLogNotificationConversions": true,
    "autoHandleNotificationDeepLinks": true,
    "silentPush": false
])
Radar.initialize(publishableKey: "prj_test_pk_...", options: options)
```

```objective-c
RadarInitializeOptions *options = [[RadarInitializeOptions alloc] initWithDict:@{
    @"autoLogNotificationConversions": @YES,
    @"autoHandleNotificationDeepLinks": @YES,
    @"silentPush": @NO
}];
[Radar initializeWithPublishableKey:@"prj_test_pk_..." options:options];
```
